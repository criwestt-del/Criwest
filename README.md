# Criwest

یه صفحه‌ی تک‌فایلی (HTML) که خودش همه‌چیز رو داره — استایل و جاوااسکریپت همه توی `index.html` هستن. هیچ سرور یا دیتابیسی لازم نداره.

## چطور بذارمش رو گیت‌هاب (GitHub Pages)

1. برو به [github.com](https://github.com) و وارد شو (یا اگه اکانت نداری، یکی بساز — رایگانه).
2. گوشه‌ی بالا سمت راست، روی **+** بزن و **New repository** رو انتخاب کن.
3. یه اسم بذار، مثلاً `criwest` — گزینه‌ی **Public** رو نگه‌دار.
4. تیک **Add a README file** رو نزن (چون خودمون یکی داریم) و روی **Create repository** بزن.
5. تو صفحه‌ی خالی ریپو، روی **uploading an existing file** بزن.
6. فایل `index.html` و `README.md` رو بکش و بنداز تو اون کادر، بعد **Commit changes** رو بزن.
7. برو به تب **Settings** ریپو، از منوی سمت چپ **Pages** رو بزن.
8. زیر **Build and deployment**، سورس رو بذار روی **Deploy from a branch**، برنچ رو `main` و پوشه رو `/ (root)` انتخاب کن، بعد **Save**.
9. چند دقیقه صبر کن؛ آدرس سایتت می‌شه:
   `https://USERNAME.github.io/criwest`
   (به‌جای USERNAME اسم اکانت گیت‌هابت رو بذار)

بعداً هر وقت خواستی چیزی رو عوض کنی، کافیه فایل `index.html` رو ویرایش‌شده دوباره تو همون ریپو آپلود کنی (یا از طریق ادیتور آنلاین گیت‌هاب مستقیم داخل مرورگر ویرایشش کنی — با زدن دکمه‌ی مداد بالای فایل).

## اگه بعداً خواستی دامنه‌ی اختصاصی بذاری

تو تنظیمات همون بخش **Pages**، قسمت **Custom domain** رو پر کن و طبق راهنمای گیت‌هاب رکوردهای DNS رو تنظیم کن.

## نکته درباره‌ی موسیقی پس‌زمینه

آهنگ «Star Shopping» از طریق پلیر رسمی و مجاز یوتیوب (embed) پخش می‌شه، نه فایل دانلودی — این‌جوری به حق کپی‌رایت احترام گذاشته می‌شه و نیازی به هاست کردن فایل صوتی نیست.
